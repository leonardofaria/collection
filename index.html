About this collection

I started using Macs when I was 12. I used to go to my brother's school after my high school classes to access (fast) Internet in Bondi Blue iMacs.

A few years later I bought an iBook 700 Mhz to create websites using Flash. A few more years, a MacBook and then many others, that I got on craigslist or from friends. All items listed in this website are functional.

About this website

I used to record the information about my collection in a GitHub gist but it was not enough.
Made this site with Tailwind CSS (I didn't write one line of CSS!), based on a Google Sheet that a service called Stein parses and gives to me. Source code of the project.
